{# ============================ WHY SHE EXISTS ============================ #}

{% if fr %}Pourquoi elle existe{% else %}Why she exists{% endif %}

{% if fr %}Les problèmes n'arrivent pas d'un coup{% else %}Problems don't arrive all at once{% endif %}

{%- if fr -%} Ils s'installent lentement. Un disque se remplit d'un pour-cent à la fois. Un SSD accumule des secteurs réalloués pendant des semaines avant de lâcher. Un service s'arrête mardi, vous le voyez vendredi. htop, smartctl et journalctl vous montrent l'instant où vous regardez. Joséphine surveille les signaux lents que vous ne regardez pas — et vous prévient pendant qu'il est encore tôt. {%- else -%} They creep in. A disk fills a percent at a time. An SSD sheds reallocated sectors for weeks before it fails. A service dies on Tuesday; you notice on Friday. htop, smartctl and journalctl show you the moment you look. Joséphine watches the slow-moving signals you don't — and speaks up while it's still early. {%- endif -%}

{% if fr %}Sans elle{% else %}Without her{% endif %} {% if fr %}le jour où ça vous rattrape{% else %}the day it catches up with you{% endif %}

  • ✕ {% if fr %}Le disque se remplit en plein build — et le build meurt à 3 h du matin.{% else %}The disk fills mid-build — and the build dies at 3 a.m.{% endif %}
  • ✕ {% if fr %}Le SSD logue des erreurs SMART depuis des semaines — vous l'apprenez quand un fichier ne s'ouvre plus.{% else %}An SSD logged SMART errors for weeks — you find out when a file won't open.{% endif %}
  • ✕ {% if fr %}Un service s'est arrêté hier ; vous le voyez quand ce qui en dépend casse.{% else %}A service stopped yesterday; you notice when something downstream breaks.{% endif %}
  • ✕ {% if fr %}Le ventilateur hurle depuis des heures sous un process emballé.{% else %}The fan has been screaming for hours under a runaway process.{% endif %}
  • ✕ {% if fr %}Les mises à jour s'accumulent depuis des mois — dont le correctif de sécurité qu'il fallait.{% else %}Updates have piled up for months — including the security fix you needed.{% endif %}

{% if fr %}Avec elle{% else %}With her{% endif %} {% if fr %}un mot discret, des jours plus tôt{% else %}a quiet word, days earlier{% endif %}

  • ● {% if fr %}« Disque à 85 % sur / et ça grimpe. Voici ce qui prend la place. »{% else %}“Disk at 85% on / and climbing. Here's what's taking the room.”{% endif %}
  • ● {% if fr %}« SMART signale des secteurs réalloués sur nvme0. Sauvegardez ce disque bientôt. »{% else %}“SMART reports reallocated sectors on nvme0. Back this disk up soon.”{% endif %}
